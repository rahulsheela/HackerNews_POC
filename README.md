## Installation

```bash
npm install
```

## Development scripts

```bash
npm run dev
```

Development Environment with hot loading functionalities

## Production scripts

```bash
npm run build
```

Builds the application and creates bundle files for productions

## To start server

```bash
npm start
```

Starts the server at`bash process.env.PORT || 3003`

## Test

```bash
npm run test
```

Runs the test files
